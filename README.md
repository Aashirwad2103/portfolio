# Aashirwad Portfolio Website

A modern and futuristic personal developer portfolio website built using **HTML, CSS, and JavaScript**.  
This project showcases professional experience, technical skills, featured projects, education, and contact information through an immersive UI with smooth animations and interactive effects.

---

## ✨ Features

- 🎨 Modern futuristic UI design
- 🌌 Animated particle background
- 🖱️ Custom animated cursor
- ⚡ Smooth scroll reveal animations
- 📱 Fully responsive layout
- 📊 Animated skill progress bars
- 🧩 Interactive project cards
- 💼 Professional experience timeline
- 🔥 Glassmorphism effects
- 🚀 Lightweight and fast performance

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Canvas API
- Intersection Observer API
- Google Fonts

---

## 📂 Sections Included

- Hero Section
- Skills & Technologies
- Work Experience
- Featured Projects
- Education
- Contact Information
